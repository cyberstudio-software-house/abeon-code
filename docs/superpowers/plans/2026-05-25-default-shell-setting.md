# Default Shell Setting Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let the user pick which shell the interactive terminal tabs spawn (default auto-detected from `$SHELL`), via Settings → "Ogólne". Only `PtyKind::Shell` is affected; Claude/Action PTYs keep using `bash`.

**Architecture:** A new `shellPath` setting (persisted via existing two-tier localStorage + SQLite). Two new Tauri commands expose `$SHELL` auto-detection and a `which`-based probe of common shells. A small Rust helper `resolve_shell` reads the persisted value with a fallback chain (settings → `$SHELL` → `"bash"`) and is the only call-site change in `pty.rs`. The frontend hydrates the dropdown from `list_available_shells` and bootstraps an auto-detected value on first launch.

**Tech Stack:** Rust (Tauri 2, `portable_pty`, ts-rs), TypeScript (React 19, Zustand, Vite), SQLite.

> **Learning-mode note:** Task 3 (Step 3) intentionally asks the user to write the `resolve_shell` body themselves. That helper encodes the fallback chain — the central UX decision of the feature.

```rust
pub fn detect_default_shell_impl() -> Option<String> {
    let raw = std::env::var("SHELL").ok()?;
    if raw.is_empty() {
```

```rust
use crate::domain::ShellInfo;

const KNOWN_SHELLS: &[&str] = &["bash", "zsh", "fish", "sh"];

fn which(name: &str) -> Option<String> {
    std::process::Command::new("which")
        .arg(name)
        .output()
        .ok()
        .filter(|o| o.status.success())
        .map(|o| String::from_utf8_lossy(&o.stdout).trim().to_string())
        .filter(|s| !s.is_empty())
}

#[tauri::command]
pub fn list_available_shells() -> Vec<ShellInfo> {
    KNOWN_SHELLS.iter()
        .filter_map(|name| which(name).map(|path| ShellInfo {
            name: name.to_string(),
            path,
        }))
        .collect()
}
```

```ts
async function bootstrapShellPath(): Promise<void> {

    if (detected && detected.length > 0) {
```
